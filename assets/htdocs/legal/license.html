License

Strova is free software. Not "free trial" free. Not "free with ads" free. Actually free.

What This Means

Strova is licensed under the GNU General Public License v3.0 (GPL v3). This isn't some corporate legal trap. It's a guarantee that Strova stays free forever.

The deal: You get full access to use, study, modify, and share Strova. In exchange, if you share modified versions, you must share the source code under the same terms. That's it.

Your Rights

The GPL gives you four essential freedoms:

  • Freedom 0: Run the program for any purpose—personal, commercial, educational, whatever
  • Freedom 1: Study how the program works and modify it (source code access required)
  • Freedom 2: Redistribute exact copies to help others
  • Freedom 3: Distribute modified versions to others, under the same license

Notice: zero restrictions on commercial use. Make money with Strova. We encourage it. Just don't try to lock up the software itself.

The Catch (There Isn't Much)

The GPL has requirements, but they're simple:

If you distribute Strova (original or modified):

  • Include the GPL v3 license text
  • Make the complete source code available
  • Keep copyright notices intact
  • Don't add restrictions beyond what GPL allows

If you modify Strova:

  • Document what you changed
  • License your modifications under GPL v3 too
  • Share source if you share binaries

Don't do this: Take Strova, modify it, sell it as "Strova Pro" without source code. That's a GPL violation and we will call you out publicly.

Distribution

You can distribute Strova however you want:

  • Burn it to CDs and hand them out on the street
  • Pre-install it on computers you sell
  • Include it in Linux distro repositories
  • Package it with your animation course

The only requirement: include the license and offer source code. For unmodified Strova, pointing to our GitHub is sufficient. For modified versions, you need to host your own source or include it.

Commercial Use

Use Strova to make:

  • Commercial animations for clients
  • Films you sell tickets to
  • YouTube videos with ad revenue
  • Assets you sell on marketplaces
  • Courses teaching animation

The GPL doesn't touch your creative output. It only covers the software itself. Your animations are 100% yours. We don't want royalties, credit, or control.

Heck, you can even sell Strova itself—charge for physical copies, support contracts, or custom builds. Richard Stallman (GPL author) is fine with this. Just follow the source code requirements above.

No Warranty

Strova is provided "as is" without warranty of any kind. This is standard for free software—we can't afford liability insurance for a project we give away.

THE SOFTWARE IS PROVIDED "AS IS", WITHOUT WARRANTY OF ANY KIND, EXPRESS OR IMPLIED, INCLUDING BUT NOT LIMITED TO THE WARRANTIES OF MERCHANTABILITY, FITNESS FOR A PARTICULAR PURPOSE AND NONINFRINGEMENT.

If Strova crashes and corrupts your project, that sucks. We feel bad. But we can't be sued for it. Use version control, save backups, and don't bet your entire studio on beta software.
